# Netflix EDA Dashboard

## Project Overview

The Netflix EDA Dashboard is an interactive web application built using Streamlit that performs Exploratory Data Analysis (EDA) on the Netflix dataset. The dashboard helps users understand content distribution, identify missing values, analyze Movies vs TV Shows, and explore country-wise content trends through visualizations and summary statistics.

This project was created to strengthen data analysis, visualization, and dashboard development skills using Python.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Streamlit
* Git & GitHub

---

## Features

### Dataset Overview

* Total number of rows and columns
* Dataset structure and summary statistics

### Missing Values Analysis

* Detect missing values in each column
* Understand data quality issues

### Movies vs TV Shows Analysis

* Count and compare Movies and TV Shows
* Visualize content distribution

### Country-wise Content Analysis

* Identify the top 10 countries with the highest number of Netflix titles
* Compare Movies and TV Shows across countries

### Interactive Sidebar Navigation

* Switch between analyses using a user-friendly sidebar
* Dynamic dashboard experience

---

## Key Insights

* Movies make up the majority of Netflix content.
* The United States contributes the largest number of titles on Netflix.
* Some metadata fields such as Director, Cast, and Country contain missing values, highlighting data quality considerations.

## How to Run

### Install Required Libraries

```bash

```

### Run the Application

```bash
streamlit run app.py
```
